Fungsi Smart Contract
inputBudget()
Admin menyimpan data biaya bulanan. Hanya bisa dipanggil oleh alamat admin yang terdaftar. Data tersimpan permanen on-chain.
getBudgetByMonth() — view
Siapapun bisa membaca data budget bulan tertentu tanpa perlu wallet. Transparan 100%.
getSPPPerSantri() — view
Mengembalikan nilai SPP per santri yang sudah dikalkulasi otomatis oleh smart contract.
getAllRecords() — view
Mengambil seluruh riwayat budget semua bulan. Digunakan untuk dashboard publik dan audit.
Panduan Penggunaan
Untuk Admin / Bendahara Pesantren
1
Instal MetaMask
Download ekstensi MetaMask di browser Chrome/Brave. Buat akun dan simpan seed phrase dengan aman.
2
Tambah Polygon Amoy Testnet
Di MetaMask, tambah network dengan Chain ID: 80002, RPC: https://rpc-amoy.polygon.technology
3
Deploy Smart Contract
Buka Remix IDE, paste kode PesantrenBudget.sol, compile dan deploy ke Amoy Testnet. Copy contract address.
4
Input Biaya Setiap Bulan
Masukkan rincian biaya di halaman Input, klik "Simpan & Rekam On-Chain", konfirmasi di MetaMask.
Untuk Wali Santri
Buka link dashboard pesantren di browser HP. Tidak perlu install apapun, tidak perlu akun. Semua data biaya bulanan terbuka untuk dibaca. Untuk memverifikasi keaslian data, klik TX Hash di setiap record — akan terbuka di Polygon Explorer yang bisa diakses siapapun di seluruh dunia.
